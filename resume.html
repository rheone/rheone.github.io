Summary

Distinguished Member of Technical Staff at Sandia National Laboratories with over twenty years of experience designing, building, and leading enterprise-scale software systems in support of national security missions for the Department of Energy (DOE) and the National Nuclear Security Administration (NNSA). Leads a portfolio of over 30 active .NET products — spanning 80-plus deployments, 3.9 million lines of code, and 8.8 billion database records that underpin laboratory-wide IT software infrastructure across networking, cybersecurity, asset management, account management, and specialized internal customer software tooling. Drives Agile team practices and DevSecOps adoption, including Software Bill of Materials (SBOM) generation and automated CI/CD pipelines aligned with federal executive mandates. Recognized Subject Matter Expert across software architecture, network infrastructure (DNS, DHCP, IPv6, IPAM), and application security; lead developer of several open-source packages and a recurring speaker at national-laboratory and DOE technical conferences.

Open-Source Projects

NetDuid

A .NET library providing a robust, strongly-typed representation of DHCP Unique Identifiers (DUIDs) as specified in RFC 8415 and RFC 6355. Supports parsing, construction, comparison, and formatting operations across .NET Standard 2.0 and modern .NET runtimes.

GitHub
C# .NET DHCP DHCPv6 RFC 8415 RFC 6355

Arcus

A .NET C# library for parsing, formatting, converting, comparing, and manipulating IPv4 and IPv6 addresses, subnets, and MAC address data formats. Handles 128-bit numbers on 32-bit platforms and supports subnet calculations across five functional areas.

GitHub
C# .NET IPv4 IPv6 Networking Subnetting

Gulliver

A .NET C# utility package for bitwise manipulation of arbitrary-length byte arrays as numerical objects, with full support for variable-length storage and endianness. Provides boolean algebraic operations, bitwise shifting, and unsigned endian-aware arithmetic.

GitHub
C# .NET Bitwise Byte Arrays Endianness

Chordly

A JavaScript library for keyboard-driven user experience and web accessibility in browser-based applications. Enables detection and handling of key sequences, chord combinations, and Konami-style input patterns.

GitHub
JavaScript Keyboard Accessibility UX
