Open-Source Projects

Software developed at Sandia National Laboratories and released under the Apache License 2.0.

NetDuid

A .NET library providing a robust, strongly-typed representation of DHCP Unique Identifiers (DUIDs) as specified in RFC 8415 and RFC 6355. Designed for network-facing applications that need to parse, construct, compare, and format DUIDs when working with DHCPv6 infrastructure, logging, or testing. Targets .NET Standard 2.0 and modern .NET runtimes with no runtime dependencies beyond the base class library.

GitHub
C# .NET Standard 2.0 DHCP DHCPv6 RFC 8415 RFC 6355 Apache 2.0

Arcus

A .NET C# library for calculating, parsing, formatting, converting, and comparing IPv4 and IPv6 addresses, subnets, and MAC address data formats. Handles 128-bit numbers on 32-bit platforms and provides functionality across five areas: Types, Comparers, Converters, Math, and Utilities. With over 79,000 NuGet downloads, Arcus is widely used in network-facing applications across the .NET ecosystem.

GitHub
C# .NET Standard 2.0 IPv4 IPv6 MAC Address Networking Subnetting Apache 2.0

Gulliver

A .NET C# utility library for the bitwise manipulation of arbitrary-length byte arrays as numerical objects, with full support for variable-length storage and endianness. Provides a standard set of boolean algebraic operations, bitwise shifting, and unsigned endian-aware arithmetic. Originally created to support Arcus, it has since grown into a standalone package with over 77,000 NuGet downloads.

GitHub
C# .NET Standard 2.0 Bitwise Byte Arrays Big Endian Little Endian Math Apache 2.0

Chordly

A JavaScript library for keyboard-driven user experience and web accessibility in browser-based applications. Enables detection and handling of key sequences, chord combinations, and sequential hotkey patterns. Useful for building keyboard-navigable interfaces, adding power-user shortcuts, and improving accessibility in web applications — without a framework dependency.
