Open-Source Projects

Software developed at Sandia National Laboratories and released under the Apache License 2.0.

NetDuid

A .NET library providing a robust, strongly-typed representation of DHCP Unique Identifiers (DUIDs) as specified in RFC 8415 and RFC 6355. Designed for network-facing applications that need to parse, construct, compare, and format DUIDs when working with DHCPv6 infrastructure, logging, or testing. Targets .NET Standard 2.0 and modern .NET runtimes with no runtime dependencies beyond the base class library.

dotnet add package NetDuid

MacSharp

A .NET micro library providing a robust, strongly-typed representation of MAC addresses. It includes EUI-48 and EUI-64 formats as specified in IEEE 802 and RFC 7042. Designed for that need to parse, construct, compare, and format MAC addresses. Targets .NET Standard 2.0 and modern .NET runtimes with no runtime dependencies beyond the base class library.

C# .NET Standard 2.0 MAC Address EUI-48 EUI-64 IEEE 802 RFC 7042 Apache 2.0

Arcus

A .NET C# library for calculating, parsing, formatting, converting, and comparing IPv4 and IPv6 addresses, subnets, and MAC address data formats. Handles 128-bit numbers on 32-bit platforms and provides functionality across five areas: Types, Comparers, Converters, Math, and Utilities. With over 79,000 NuGet downloads, Arcus is widely used in network-facing applications across the .NET ecosystem.

dotnet add package Arcus
GitHub
C# .NET Standard 2.0 IPv4 IPv6 MAC Address Networking Subnetting Apache 2.0

Gulliver

A .NET C# utility library for the bitwise manipulation of arbitrary-length byte arrays as numerical objects, with full support for variable-length storage and endianness. Provides a standard set of boolean algebraic operations, bitwise shifting, and unsigned endian-aware arithmetic. Originally created to support Arcus, it has since grown into a standalone package with over 77,000 NuGet downloads.

dotnet add package Gulliver
GitHub
C# .NET Standard 2.0 Bitwise Byte Arrays Big Endian Little Endian Math Apache 2.0

Chordly

A JavaScript library for keyboard-driven user experience and web accessibility in browser-based applications. Enables detection and handling of key sequences, chord combinations, and sequential hotkey patterns. Useful for building keyboard-navigable interfaces, adding power-user shortcuts, and improving accessibility in web applications — without a framework dependency.

Government Use Notice Projects

Not all software created at Sandia reaches a public release. The Government Use Notice (GUN) process is a DOE/NNSA mechanism that allows national laboratories to distribute software internally across the federal ecosystem — to peer laboratories, agencies, and authorized contractors — under restricted terms. GUN software is developed with government funding, serves government mission needs, and remains within the government distribution network rather than being published under a public open-source license.

Government Use Notice — Not publicly available

Reliable Automated File Transfer Service (RAFTS)

A RESTful API-driven web interface service for transferring large data sets across multiple classified and unclassified network security domains within the DOE/NNSA ecosystem. The suite includes Angular web interfaces, multiple .NET C# projects, and automated processing scripts supporting cross-boundary data movement. Later renamed TIDE.

C# .NET REST API Angular Cross-Domain Security Domains
Government Use Notice — Not publicly available

Mobile Credential Manager (MCM)

An ASP .NET web application enabling laboratory workforce members to self-manage derived Personal Identity Verification (PIV) mobile credentials — as specified under Homeland Security Presidential Directive 12 (HSPD-12) — for authentication to Sandia resources.
